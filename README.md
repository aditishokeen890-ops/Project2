# Project2
### Problem Statement 
Insurance companies often struggle to track and analyze large volumes of policy and claims data. Key challenges include:

Identifying which policy types generate the most revenue
Understanding claim patterns and high-risk customer segments
Monitoring policy activity (active vs inactive)
Improving claim settlement efficiency

This project aims to solve these challenges by building an interactive dashboard that provides clear insights into premiums, claims, customer demographics, and policy performance.

### Project Overview 
This project focuses on analyzing insurance data to uncover trends in premiums, claims, policy types, and customer demographics. The dashboard provides actionable insights for business decision-making.

### Tools and Technologies
Power BI (Data Visualization)
 MS SQL (Data Extraction & Transformation)
Python (Data Cleaning & Analysis)
Excel (Preprocessing)
